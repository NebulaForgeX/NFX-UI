# Animations and loaders

All animation and loading components are imported by name from **`nfx-ui/animations`**; no separate CSS import.

**Note:** `PixelBlastBackground` is **not** in this barrel — use [`nfx-ui/pixel-blast`](../pixel-blast/README.md) instead.

---

## Component list

| Component | Description | Doc |
|-----------|-------------|-----|
| ECGLoading | ECG-style loader | [ecg-loading.md](./ecg-loading.md) |
| TruckLoading | Truck-style loader | [truck-loading.md](./truck-loading.md) |
| BounceLoading | Bouncing square/circle loader | [bounce-loading.md](./bounce-loading.md) |
| WaveBackground | Wave background (theme-aware) | [wave-background.md](./wave-background.md) |
| SquareBackground | Grid square background | [square-background.md](./square-background.md) |
| LetterGlitchBackground | Letter glitch background | [letter-glitch-background.md](./letter-glitch-background.md) |

---

## Import example

```tsx
import {
  ECGLoading,
  TruckLoading,
  BounceLoading,
  WaveBackground,
  SquareBackground,
  LetterGlitchBackground,
} from "nfx-ui/animations";
```

# 动效与加载

动效与加载组件从 **`nfx-ui/animations`** 按名引入，无需单独引入 CSS。

**说明：** `PixelBlastBackground` **不在** 此 barrel 中 — 请使用 [`nfx-ui/pixel-blast`](../pixel-blast/README.md)。

---

| 组件 | 说明 | 文档 |
