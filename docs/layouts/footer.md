# Footer

Footer for copyright, links, etc.

---

## Import

```tsx
import { Footer } from "nfx-ui/layouts";
```

---

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| children | ReactNode | No | — | Footer content. |

---

## Input / Output

- **Input:** children.
- **Output:** Renders footer.

---

## Example

```tsx
<Footer>
  <span>© 2025 NFX</span>
</Footer>
```


---

## 参数

| 参数 | 类型 | 必填 | 默认 | 说明 |
|------|------|------|------|------|

## 输入 / 输出

- **输入：** children。
- **输出：** 渲染底栏。
